### 1创建项目
```bash
npx create-react-app react-admin_client
```

### 2启动项目
```bash
cd react-admin_client
npm start
```
### 3打包发布
```bash
npm run build
npm install -g serve
server build
```

### 基本结构
src
 --api ajax相关
 --assets 共用资源
 --components 非路由组件
 --config 配置
 --pages 路由组件
 --utils 工具模块
 --App.js 程序根组件
 --index.js 入口js
